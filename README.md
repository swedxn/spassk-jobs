# Работа рядом — Спасск-Дальний

Бесплатный статический агрегатор вакансий с жёсткой геофильтрацией. Основной список принимает только вакансии, где явно указан Спасск-Дальний. Другие города, общерегиональные предложения, вахта и переезд не показываются.

## Архитектура

- сайт: GitHub Pages (бесплатно для публичного репозитория);
- облачное хранилище: JSON/CSV в GitHub-репозитории и CDN GitHub Pages;
- обновление: GitHub Actions раз в 3 часа;
- источники: HeadHunter, «Работа России», FarPost, Rabota1000, «ЦентрРабота», публичные Telegram-каналы и проверенные страницы работодателей;
- трекер откликов: `localStorage`, экспорт/импорт JSON;
- история рынка: `data/history.json` и `data/changes.json` с новыми, изменившимися и исчезнувшими вакансиями;
- два независимых рейтинга: Fit Score (подходит человеку) и Opportunity Score (насколько срочно стоит посмотреть);
- зависимости: отсутствуют, нужен только Node.js 20.

## Локальная проверка

```bash
npm test
npm run import
npm run serve
```

Сайт откроется на `http://127.0.0.1:43871`.

При сетевой блокировке импорт не создаёт моки: он публикует последний вручную проверенный реальный срез и записывает причину в `data/import-report.json`.

## Бесплатные лимиты

Для публичного репозитория стандартные GitHub Actions и GitHub Pages подходят без платных сервисов. Планировщик GitHub может запускаться с задержкой при высокой нагрузке; импорт идемпотентен и следующий запуск восстановит график.
