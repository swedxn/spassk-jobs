# Работа рядом — Спасск-Дальний

Бесплатный статический агрегатор вакансий с жёсткой геофильтрацией. Основной список принимает только вакансии, где явно указан Спасск-Дальний. Другие города, общерегиональные предложения, вахта и переезд не показываются.

## Архитектура

- сайт: GitHub Pages (бесплатно для публичного репозитория);
- облачное хранилище: JSON/CSV в GitHub-репозитории и CDN GitHub Pages;
- обновление: GitHub Actions раз в 3 часа;
- источники: официальный API HeadHunter, два официальных запроса «Работы России» (городской КЛАДР и текстовый контроль), FarPost, ГдеРабота, Rabota1000, «ЦентрРабота», публичные Telegram-каналы и прямые страницы работодателей, включая Спасский завод ЖБИ;
- трекер откликов: `localStorage`, экспорт/импорт JSON;
- история рынка: `data/history.json` и `data/changes.json` с новыми, изменившимися и исчезнувшими вакансиями;
- два независимых рейтинга: Fit Score (подходит человеку) и Opportunity Score (насколько срочно стоит посмотреть);
- интерфейс: React + Vite, Motion и GSAP; сборка остаётся полностью статической и публикуется на GitHub Pages;
- доступность: клавиатурная навигация, семантическая разметка и автоматическое отключение сложной анимации при `prefers-reduced-motion`.

## Локальная проверка

```bash
npm test
npm run audit:data
npm run import
npm run dev
npm run build
```

Сайт откроется на `http://127.0.0.1:5173/spassk-jobs/`.

При сетевой блокировке импорт не создаёт моки: он временно сохраняет последний успешно полученный реальный срез и записывает причину в `data/import-report.json`. Непроверяемые снимки автоматически истекают через 14 дней, история закрытых карточек очищается через 120 дней. Если успешная проверка источника больше не находит вакансию, она исчезает из активного списка при этом обновлении.

Команда `npm run audit:data` проверяет всю публикуемую базу: точный город, отсутствие вахты и переезда, корректность дат и URL, уникальность идентификаторов и предупреждений, чистоту FarPost-адресов, согласованность статистики и экспортов. Такой же аудит входит в тесты перед облачным деплоем.

Полностью удалённые вакансии не смешиваются с городскими и показываются отдельным блоком. Для сайтов с CAPTCHA, авторизацией или запретом в `robots.txt` обход защит не применяется: ссылка остаётся в каталоге источников для ручной проверки.

## Бесплатные лимиты

Для публичного репозитория стандартные GitHub Actions и GitHub Pages подходят без платных сервисов. Планировщик GitHub может запускаться с задержкой при высокой нагрузке; импорт идемпотентен и следующий запуск восстановит график.
